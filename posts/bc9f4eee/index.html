操作过程中的报错

  1. github 仓库无法使用

原因：.deploy_git 文件下没有初始化.git仓库

解决方法： git init ./.deploy_git/

  1. SSL 配置

有两个地方：

  1. 个人账户的SSL key配置，这个是全局的；入口：Settings –> SSH and GPG Keys
  2. 对应仓库的 SSL key配置，这个是针对特定仓库的；入口：repo –> Settings –> Deploy Keys
