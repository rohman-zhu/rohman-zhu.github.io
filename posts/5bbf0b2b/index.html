防火墙设备

Posted on 2018-12-05 Edited on 2024-12-22

前言

最近接触思科的设备，主要以交换机和防火墙居多。因此就记录一下相关参数。

对于防火墙性能评定：

  1. 网络层的访问控制：
    1. 对于 IP/MAC 绑定功能，防火墙是否能自动搜集 。
    2. NAT（网络地址转换）
  2. 应用层上的控制：
    1. 是否提供 HTTP 协议的内容过滤
    2. 是否提供 SMTP 协议内容过滤，防止邮件攻击
    3. 是否提供 FTP 协议内容过滤
  3. 管理和认证：
    1. 基于 WEB 界面的 WUI
    2. 基于图形界面的 GUI
    3. 基于命令行的 CLI
  4. 审计和日志以及存储方式
