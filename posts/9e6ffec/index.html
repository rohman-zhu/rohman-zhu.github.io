Windows客户端加域后没有产生DNS记录

Posted on 2025-06-18 Edited on 2025-06-22 In Windows Server

故障描述

一些已经加域的客户端，该客户端的主机名无法通过DNS解析到IP；但客户端的加域状态是正常的。

例如：客户端的主机名为 client-demo.domain.com

原因分析

DNS 注册失效：可能因为网络延迟 或者 服务同步问题导致。
