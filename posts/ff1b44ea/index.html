Powershell的Invoke-RestMethod实例

Posted on 2026-02-14 In Windows Server

背景

最近看到同事写的Python脚本去调去网页信息，感觉写得有点复杂，所以就想看看能不能转义成Powershell脚本；

核心功能诉求：

  1. 根据特定条件去检索 Atlassian JIRA 的工单信息；
  2. 对工单的流量进行流转；
  3. 对工地添加自定义备注信息；

实操

登录模块

在两个不同版本的Powershell中，鉴权方式不太一样，Powershell 5 要麻烦一些，但要更通用一些；而Powershell 7 方便很多，当然也需要用户自行升级；

本次使用的Jira鉴权可以用base64编码提交，以下两个版本对应的指令是不一样的：

HTTP Basic 认证是一种简单的身份验证方式，客户端在请求头中携带 Authorization: Basic <凭证>，其中 <凭证> 是 用户名:密码 的 Base64 编码字符串。

检索模块

对于检索使用的是JQL（Jira Query Language），如果全部检索词都是英文，整体运行不会有太大问题；但只要用了中文且还是powershell 55，jql中包含中文那就是一个麻烦事，请求时会报400，我排查了很久最后才发现这个问题；powershell 7 中运行是不会有问题的；

日期格式要求符合ISO 8601，yyyy-MM-ddTHH:mm:ssZ
传输给-Body的参数系统默认会使用 ANSI编码(如GBK)转成字节流，导致中文字符乱码，传输字节数组就可避免此问题；
Powershell5.1默认使用的是ANSI编码，而Powershell7默认使用的是UTF8编码；使用 [System.Text.Encoding]::Default 可以看到编码信息；
jql用于检索的中文名，需要用引号，否则会报错;
