Powershell脚本-网页爬虫实例

Posted on 2026-07-18 In Windows Server

原始需求

有一个网址上面的链接会定期发生变更，需要写一个程序去及时获取最新的网页链接，需求拆解：

  1. 能够访问到目标网页，获取到网页上的内容；
  2. 对上面提供的链接，能够递归读取；
  3. 将结果记录下来；

实现方案

假定要访问的地址是 https://www.test.com

如何模拟浏览器去访问

打开Chrome浏览器，按F12进入调试界面，看到Network页，找到对应所需请求的页面，比如 www.test.com 项 看到有如下信息

如果直接使用 Invoke-WebRequest -Method “GET” -UseBasicParsing -Uri $URI ，不包含请求头，返回的值不是所需内容，容易有屏蔽情况；因此需要额外增加请求头，模拟我们用网页正常访问。

解析内容

获取到的HTML文本内容后，需要做解析;

第一部分是关于截取文本内容，查询到有两种方案，第一种是解析一整串字符串，第二种是逐行显示的文本文件：
这里选用第一种方案，适配性比较高，而我们获取到的文本并不会保存文件，而是一个字符串参数；

第二部分是对截取的 HTML 文本做链接解析 ，截取出我们真正需要的，这里就用到powershell中的正则表达式了；

正则表达式 ‘href=”([^”]*)”‘ 的含义拆解

正则部分 含义
href=” href=”（普通文本）
( 开始捕获（我们要提取的内容）
[^”]* 匹配任意数量的非双引号字符（即链接内容，直到遇到双引号为止）
) 结束捕获
“ 匹配闭合的双引号

找到 href=”，然后把后面直到下一个 “ 之间的所有内容提取出来。
