No.5йқһеёёйҮҚиҰҒпјҢйҒҝе…Қиө„жәҗжөӘиҙ№пјҢз”ЁжҲ·д№ҹи®ёдёҚжё…жҘҡиҮӘе·ұйңҖиҰҒд»Җд№ҲпјҢжӣҙеӨҡжғ…еҶөдёӢе°ұжҳҜд»Җд№Ҳиө„жәҗйғҪдјҡз”іиҜ·пјҢиҝҷж— з–‘дјҡеўһеҠ иҝҗз»ҙдәәе‘ҳдёҺж”ҜжҢҒдәәе‘ҳзҡ„е·ҘдҪңйҮҸгҖӮ

дҝЎжҒҜе®үе…Ёдҫ§иҜ„дј°

йңҖиҰҒжҳҺзЎ®дҝЎжҒҜе®үе…ЁзәўзәҝпјҢзЎ®дҝқж•°жҚ®иҝӣеҮәж–№еҗ‘пјҢеҜ№дәҺж•Ҹж„ҹзӯ–з•ҘдёҖе®ҡиҰҒи®ҫзҪ®жңүзӣ‘жҺ§гҖӮжүҖжңүзҡ„йғЁзҪІж–№жЎҲгҖҒйҳІзҒ«еўҷзӯ–з•Ҙз”іиҜ·еә”жңүе®Ўжү№жҠҘеӨҮгҖӮ

ж•ҲзҺҮжҖ»з»“

з®ЎзҗҶдҫ§

еҜ№дәҺ VMware Horizon View дёӯзҡ„ RDS жЎҢйқўйғЁзҪІпјҢеҹәжң¬дёҠйғҪжҳҜеңЁChromeжөҸи§ҲеҷЁдёӯз”Ёйј ж ҮзӮ№еҮ»жҺҲжқғпјҢиҝҷдёӘйҮҚеӨҚжҖ§зҡ„ж“ҚдҪңеҸҜд»Ҙз”ЁPython3+Selenuim+ChromeжқҘеҒҡиҮӘеҠЁеҢ–зӮ№еҮ»йғЁзҪІгҖӮ

иө„жәҗдҫ§

еҜ№дәҺеҲқе§ӢеҢ–RDSHжңҚеҠЎеҷЁпјҢеҸҜд»ҘдҪҝз”Ё Windows + Openssh + Ansible жқҘжү№йҮҸйғЁзҪІгҖӮ

RDSHжңҚеҠЎеҷЁеқҮдёәWindowsжңҚеҠЎеҷЁпјҢеңЁдёҠйқўе®үиЈ…OpensshпјҢе№¶й…ҚзҪ®е…Ғи®ёAnsibleиҙҰжҲ·жқҘйғЁзҪІгҖӮ

Ansibleдҫ§еҲҷжҳҜйңҖиҰҒеҶҷеҘҪзӣёеә”зҡ„и„ҡжң¬жү№йҮҸдёӢеҸ‘еҚіеҸҜпјҢиҝҷйҮҢиҰҒжіЁж„ҸAnsibleжңҚеҠЎеҷЁзҡ„й…ҚзҪ®пјҢйҒҝе…Қи·‘зҡ„д»»еҠЎеӨӘеӨҡеҜјиҮҙCPUзҲҶж»ЎгҖӮ

еҹәзЎҖжһ¶жһ„дҫ§

еә•еұӮиө„жәҗиҰҒе……и¶іпјҢеңЁзҙ§жҖҘж—¶еҖҷиғҪеӨҹжҸҗдҫӣжүҖйңҖиө„жәҗпјҢиҝҷдёӘйңҖиҰҒдёҺдёҡеҠЎж–№е•Ҷи°ҲпјҢи®©е…¶еҒҡзӣёе…ійў„з®—гҖӮ
